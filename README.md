# Proiect de licență — Molecular

**Molecular** is a real-time molecular and subatomic physics simulator built on a
custom C++ / OpenGL engine.

- **2D molecular dynamics** — atoms (H, O, C, N) interacting via Lennard-Jones
  and Coulomb forces, with covalent bonding, multiple integrators, and energy
  tracking.
- **3D subatomic** *(in progress)* — protons / neutrons / electrons under
  Coulomb and the Yukawa strong-force approximation.
